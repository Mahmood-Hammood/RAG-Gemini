## RAG Application using Gemini Pro

This project is a RAG software that answers questions about a specific research paper. The main idea is:

1- Load the content of a PDF research paper and split it into smaller chunks.

2- Convert these chunks into embeddings.

3- Use the Gemini Pro model to generate answers based on the retrieved information.

4- When a question is asked, the software first finds the most relevant parts of the paper and then provides a concise answer.
